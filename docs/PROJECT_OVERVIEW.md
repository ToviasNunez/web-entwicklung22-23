# Project Overview

## Purpose
TravelLens ist eine Fullstack-Webanwendung zum Teilen, Bewerten und Entdecken von Reiseerfahrungen. Ziel ist es, strukturierte Reiseberichte und Tipps bereitzustellen.

## Scope
**Enthalten:**
- Angular-Frontend (SPA)
- Node.js/Express/MongoDB-Backend
- Authentifizierung (JWT)
- Post-Management (CRUD)
- Demo-Modus ohne Datenbank
- AI-Governance-Layer

**Ausgeschlossen:**
- Native Mobile-Apps
- Nicht-öffentliche APIs

## Key Components
- **Frontend:** Angular 15, Material, Bootstrap, Auth, Post-Module
- **Backend:** Express, Mongoose, Auth, Post/User-API, Demo-Daten
- **Infrastruktur:** .env, AI-Governance, Scripts

## Architecture Summary
Das System besteht aus einem Angular-Frontend (SPA) und einem Express/MongoDB-Backend. Die Kommunikation erfolgt über REST-APIs. Ein Demo-Modus ermöglicht Betrieb ohne Datenbank.
